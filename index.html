About E4Warning Vector Risk Portals

E4Warning Vector Risk Portals is an interactive platform for visualizing and analyzing the risk of disease-vector mosquitoes. The risk information displayed is generated by predictive models running in companion repositories. The Vector Risk Portals site is being developed under the E4Warning project (Eco-Epidemiological Intelligence for early Warning and response to mosquito-borne disease risk in Endemic and Emergence settings) funded by the European Union's Horizon Europe programme under Grant Agreement 101086640.

Data Sources

The platform accesses risk assessment data from public URLs, including:

  • CSV files containing time-series risk data
  • GeoJSON files with spatial risk information
  • GeoTIFF files for raster-based risk maps

Available Regions

  • Countries: Spain, Brazil
  • Cities: Barcelona, Rio de Janeiro

Technology

This is a static web application built with modern web technologies and served via GitHub Pages. It uses Leaflet for interactive mapping and Chart.js for data visualization.
